# Project Skill Overrides

Drop project-specific agents and helpers here. Codex will load this directory alongside the global skills once the template is bootstrapped.

## Suggestions
- Add a descriptive `AGENTS.md` when a project needs custom behavior.
- Keep overrides versioned so teammates can reproduce the same setup.
- If you borrow skills from the toolkit, reference the global `codex/` files and only override the fields you own.
